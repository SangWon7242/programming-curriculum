1. **주요 대상**

- 새로운 기술 스택을 탐색하는 현직자
- 현 학원 정규 과정 중 웹 수업을 진행하고 있는 수강생

2. **수강생 니즈**

```
✓ 최신 기술 트렌드 파악

✓ 향후 학습 방향성 결정
✓ 프론트엔드 적성 확인
```
